# Backend

# Run le backend

## 1 - Run with Docker

Depuis la racine du projet, lancez `docker compose up`.

## 2 - Run with MongoDB Docker Image & on terminal

- Lancer l'image de Docker MongoDB
- N'oubliez pas de `npm install` dans le dossier backend si pas déjà fait.

Puis
```
cd ./backend
node server.js
```
